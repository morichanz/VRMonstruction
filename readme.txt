[作品の内容]
 自分が怪獣になって3分間全力で街を破壊します。
 破壊した建物や戦車、戦闘ヘリ、戦艦によって被害額が増えます。
 残り時間が0になると、ボス戦となりメカ怪獣と戦います。
 メカ怪獣との戦闘はフェーズごとに決まっており、敵のパンチをはじくフェーズ、
 ミサイルをはじくフェーズ、メカ怪獣とプレイヤーが同時にビームを出すフェーズ、
 そして、最後にボスを破壊するフェーズとなっています。
 ボス撃破でクリアとなりリザルト画面に移ります。
 破壊総額に応じてランキングが表示されます。

[担当箇所]
 戦艦の実装全般、戦艦の弾の弾道エフェクト、着弾時の爆発エフェクトを実装しました。
 戦艦のテストシーン：Assets->WorkSpace->Morita->Scene->BattleShipMove
 マージシーン：Assets->WorkSpace->Oyama->Scene->TestStage

[こだわり]
＜戦艦＞
 ～移動～
　　　なるべくスムーズに移動できるように、leapを使い滑らかな軌道を描くように実装しました。
　　　3次ベジェ曲線を利用し、半円を描くように移動しています。
　　　ベジェ曲線を利用した理由は、実装にそこまで時間がかからなさそうだったことと、開始地点、
　　　中間地点を自由に変更できるからです。

 ～戦艦、弾の数管理～
　　　戦艦の数や、弾はオブジェクトプールを利用しています。
　　　また、戦艦の数は自由に変更できるようにしており、何艘出すかはプランナーが自由に決められ
　　　ます。
 
＜弾＞
 弾が発射されたときの弾道エフェクトも違和感のないようにパーティクルシステムを利用し
 色の調整から煙の演出まで自作しました。
